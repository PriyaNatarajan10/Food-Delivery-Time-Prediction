# Food Delivery Time Prediction 🚀

## 📌 Project Overview
This project predicts food delivery time using machine learning.

## 🎯 Objective
To build a model that estimates delivery time based on factors like distance, traffic, and ratings.

## 📊 Dataset
Dataset taken from Kaggle with 40,000+ records.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 🤖 Model Used
Linear Regression

## 📈 Result
Achieved MAE of around 6.6 minutes.

- Use advanced ML models
